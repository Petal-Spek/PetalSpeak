# 🌸 Petal Speak

**Petal Speak** on veebiprojekt, mille eesmärk on populariseerida lillede keelt ning aidata inimestel väljendada oma emotsioone läbi floristika. Platvorm pakub mugavat ja kaasaegset viisi valida lillekimp, mis peegeldab saatja tundeid.

## 📌 Projekti kirjeldus

Veebileht ühendab **interaktiivse testi** ja **veebipoe**.

Kasutajad vastavad mitmele küsimusele, mille põhjal süsteem määrab nende emotsioonid ning soovitab sobivaid lilli või valmis kimpe.

Lisaks saavad kasutajad:

- sirvida kimpude kataloogi  
- tutvuda erinevate lillede tähendustega  
- valida kompositsioone erinevateks sündmusteks  
- tellida kimpe veebis

## 🎯 Projekti eesmärk

Projekti eesmärk on aidata inimestel väljendada oma tundeid **teadlikumalt ja sümboolsemalt**.

Paljud inimesed soovivad kinkida lilli, kuid ei tea, millised lilled sümboliseerivad kindlaid emotsioone.

**Petal Speak** muudab selle protsessi:

- lihtsaks  
- harivaks  
- isikupäraseks

## 💡 Miks see on oluline

Platvorm on kasulik:

- klientidele, kes otsivad sobivat kingitust  
- inimestele, kes soovivad rohkem teada lillede tähendustest  
- floristika populariseerimiseks laiemalt

Projekt ühendab **traditsioonilise lillede sümboolika** **kaasaegse digitaalse lahendusega**, muutes kimbu valimise mugavamaks, informatiivsemaks ja emotsionaalselt väärtuslikumaks.

## 🌷 Peamised funktsioonid

- Interaktiivne test emotsioonide määramiseks  
- Lillede soovitused vastuste põhjal  
- Kimpude kataloog  
- Informatsioon lillede tähenduste kohta  
- Lillekimpude veebitellimine
